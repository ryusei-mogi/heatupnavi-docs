---
title: 使い方（毎日の流れ）
---

HeatupNavi は、次の流れで使うと安定します。

1. 初回に「前提」を整える（位置・通知・単価）
2. 毎朝：おすすめ時間帯を見る
3. 夜〜翌日：前日の使用湯量を入れる
4. 週1回：生活制約や前提値を微調整する

---

## 0. はじめて使うとき（所要5〜10分）

### A. 初回セットアップ（位置/通知）

まずは [初回セットアップ](/docs/setup) を完了してください。

- 位置情報：おすすめ計算に必要な天気の取得
- 通知：毎朝7:00のリマインド（後からでも可）

### B. 設定で「電気単価（円/kWh）」を入れる

節約額や昼/夜比較の精度を上げるには、単価が重要です。

1. ホーム下部 **「設定」** を開く
2. **電気単価（円/kWh）** で、昼/夜の境界時刻を決める（1時間刻み）
3. **夜間単価** と **昼間単価** を入力

- 詳細： [設定の詳細 → 電気単価（円/kWh）](/docs/settings#電気単価円kwh)

:::tip 単価が分からないとき
まずは「だいたい」でOKです。後から正確な値に直しても、運用は崩れません。
:::

### C. 生活制約（任意）：音が気になるなら最初に設定

夜間に動作音が気になる場合は、早めに制約を入れるとおすすめが実用的になります。

- 詳細： [設定の詳細 → 生活制約](/docs/settings#生活制約沸き上げ時間の調整)

---

## 1. 毎朝やること（1分）

### 1) ホームで「おすすめ時間帯」を見る

ホームの **「おすすめ時間帯」** を確認します。

- 表示されるのは、沸き上げにおすすめの時間帯（複数のセグメントになることがあります）
- 右上に **湯切れリスク（低/中/高）** が表示されることがあります

### 2) 湯切れが心配なら「安全側（湯切れ対策）」を優先

湯切れリスクが高い日は、ホームカード内に **安全側** の時間帯が表示されます。

- 「今日は使用が多そう」
- 「来客がある」
- 「冷え込みが強い」

といった日は、安全側を選ぶのがおすすめです。

### 3) 必要なら「詳しく見る」で根拠を確認

カードをタップすると、
- 必要な沸き上げ時間
- おすすめの理由
- データ状態（入力不足など）

が確認できます。

## 2. 不在の日は「沸き上げ停止」を使う

旅行・出張などで不在のときは、ホーム上部の **「沸き上げ停止」** をONにします。

- その日の **沸き上げ計算と通知を停止**します
- **翌日には自動的に再開**します

:::caution 使いどころ
「今日は絶対にお湯を使わない」前提の機能です。
少しでも使う可能性がある日は、停止せず「安全側」を選ぶほうが無難です。
:::

---

## 3. 夜〜翌日にやること（1分）：前日の使用湯量を入力

ホーム下部の **「前日の使用湯量」** を開き、前日分を入力します。

### 入力のしかた（簡単）

- **少なめ / ふつう / 多め** をタップ（プリセット）
- `+10 / -10` で微調整

### 「1日だけ増やす」（イベント日に便利）

来客・入浴回数増などで使用量が増えた日は、

- `+50` / `+100` / `+150` をタップ

で“その日だけ”上乗せできます。

### 数値を直接入力したい

「詳細入力（キーボード）」を開いて、Lを直接入力できます。

:::tip 入力できない日があってもOK
使用湯量が未入力の場合は、アプリが「標準量」や「直近平均」などで推定して計算します。
ただし、実績を入れた方が安定しやすいです。
:::

## 4. 週1回の見直し（5分）：おすすめが合わないときの調整順

「おすすめが出ない」「湯切れが不安」「逆に余る」などのときは、次の順で1つずつ調整します。

### 優先度1：生活制約（候補が出ない/時間が合わない）

- 使える時間を減らしすぎていないか
- 避けたい時間を長くしすぎていないか
- 最大分割数が少なすぎていないか

→ 詳細： [設定の詳細 → 生活制約](/docs/settings#生活制約沸き上げ時間の調整)

### 優先度2：電気単価（昼/夜の判断が直感とズレる）

- 昼/夜の境界時刻が実態と合っているか
- 単価の入力が極端になっていないか

→ 詳細： [設定の詳細 → 電気単価（円/kWh）](/docs/settings#電気単価円kwh)

### 優先度3：使用湯量（湯切れ/余りの体感とズレる）

- 実績を数日分入れて、平均が安定するか確認
- イベント日だけ増やす（+50/+100/+150）を使う

### 優先度4：プロ設定（上級者向け）

計算上の前提を変えると、おすすめ時間（必要運転時間）が変わります。

- タンク容量
- 貯湯温度（計算用）
- 給水温（自動推定/手動）
- 保温ロス

→ 詳細： [設定の詳細 → プロ設定](/docs/settings#プロ設定上級者向け運転時間の見積り)
