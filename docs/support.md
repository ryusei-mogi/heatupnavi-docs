## 送ってほしい情報（コピペ用テンプレ）

不具合調査をスムーズにするため、分かる範囲で次を添えてください。

```text
【内容】質問 / 不具合 / 改善要望
【やりたかったこと】
【起きたこと（表示された文言があればそのまま）】
【再現手順】（例：1. ホームで更新 → 2. おすすめ詳細を開く → 3. ...）
【発生頻度】毎回 / ときどき / 1回だけ
【発生日時】だいたいでOK

【端末】iPhone機種名
【iOSバージョン】
【アプリバージョン】（分かれば）

【設定の状況】（分かる範囲で）
- 通知：ON/OFF
- 位置：取得済み/未取得
- 電気単価：入力済み/未入力
- 生活制約：あり/なし
- ソーラー：ON/OFF

【添付】スクリーンショット / 画面収録 / CSV（必要に応じて）
```

:::caution 個人情報の取り扱い
スクリーンショットやCSVには、生活パターンが推測されうる情報が含まれる可能性があります。
共有前に不要な箇所はマスクしてください。
:::

---

## よくある相談（例）

- 湯切れが不安で設定を変えられない
- おすすめ時間帯が生活に合わない
- 推定節約額が出ない（入力不足）
- 通知が届かない（毎朝7:00）
- 生活制約を入れたらおすすめが出なくなった

## 返信までの目安

内容によっては確認に時間がかかる場合があります。
緊急性が高い場合は、その旨を先頭に書いてください。
